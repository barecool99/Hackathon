1. Open terminal

2. 'venv/Scripts/activate' to initialise virtual environment

3. 'pip install -r requirements.txt' to ensure all libraries are installed/up to date.

4. 'flask run' to create the locally hosted webpage (127.0.0.1:5000)

5.  'CTRL + C' in terminal to close the flask host

PS - If you add any new libraries using pip, make sure to keep the requirements.txt updated
     using 'pip freeze > requirements.txt'
